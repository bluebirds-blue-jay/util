@bluejay/utils

Utils

General purpose utilities for Bluejay modules.

Requirements

  • node >= 7.10
  • typescript >= 2.4

Installation

npm i @bluejay/utils [--save]

Usage

import * as Utils from '@bluejay/utils`;

Utils.makeArray(1); // [1]

Using the name export.

import { Utils } from '@bluejay/utils`;

Utils.makeArray(1); // [1]

Or directly.

import { makeArray } from '@bluejay/utils';

makeArray(1); // [1]
